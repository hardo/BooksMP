## BooksMP (逗伴我读)

一个微信小程序项目，用于展示豆瓣读书用户最近在读的书目

## 怎么使用

微信搜索 `逗伴我读` 打开搜索到的小程序，或扫描下方二维码进入：

<img src="examples/code.jpg" width="400">

## 可以用来干什么

- 方便地在手机上随时随地查看自己最近想要看的书
- 追踪心爱的他（她）最近在豆瓣上的读书记录
